# KCC 60개월 창호구독 네이버용 GitHub 전체 업로드용

업로드 방법:
압축을 풀고 안에 있는 전체 파일/폴더를 GitHub 저장소 루트에 그대로 올리세요.

업로드 후 주소:
https://lifeinfo24.kr/nwindow/

구성:
- index.html : 루트 접속 시 /nwindow/ 이동
- nwindow/ : 네이버용 메인 랜딩
- privacy/ : 개인정보처리방침
- terms/ : 이용안내

반영:
- 원본 HTML/CSS/JS/이미지 구조 유지
- 초기 팝업 제거
- PC 우측 floating 상담 버튼 유지
- 상담 버튼 클릭 시 상담폼 영역으로 스크롤 이동
- 폼 제출 시 https://www.albarich.com/pt/wVfRT8GAFk 연결
- 기존 하단 사업자정보 제거
- 허브153 하단 정보 적용
- footer는 원본 .contents 축 기준 중앙정렬
