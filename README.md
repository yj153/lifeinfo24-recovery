## 업로드 방법
기존 GitHub 저장소 루트에 아래 4개를 올리세요.
- index.html
- nhubrecovery/
- privacy/
- terms/

기존 index.html/privacy/terms를 덮어쓰기 싫으면 `nhubrecovery/`만 올려도 페이지 자체는 열리지만, 하단 개인정보/이용안내 링크까지 확인하려면 privacy/terms도 같이 올리는 것을 권장합니다.
