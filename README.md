# krecovery - 법무법인 서앤율 카카오용 독립 랜딩

## 업로드 방법
GitHub 저장소에서 기존 krecovery 폴더를 삭제한 뒤, 이 ZIP 안의 krecovery 폴더를 그대로 루트에 업로드하세요.

선택:
- 루트 index.html도 함께 올리면 메인 접속 시 krecovery로 이동합니다.

## 접속 주소
https://lifeinfo24.kr/krecovery/

## 상담신청 흐름
/krecovery/
→ /krecovery/apply/
→ 알바리치 입력폼 iframe 바로 표시

## 주의
전화번호는 제거되어 있습니다.
이메일은 유지되어 있습니다.
'무료' 문구는 제거되어 있습니다.

## 본문 이미지 확대 + CTA 3개로 축소
- 본문 CTA를 상·중·하 3개만 남겼습니다.
- 이미지 안에 박힌 작은 글씨가 더 크게 보이도록 PC 최대 표시 폭을 1280px로 확대했습니다.
- 모바일은 이미지 표시를 확대해 작은 글씨가 덜 작게 보이도록 조정했습니다.
- '무료' 문구, 전화번호는 제거 상태 유지.

## 강한 확대 버전
- 이미지 안에 박힌 글씨는 CSS 폰트로 커지지 않으므로 이미지 자체를 강하게 확대했습니다.
- PC 최대 이미지폭 1600~1700px로 확대했습니다.
- 모바일은 텍스트가 보이도록 175~195%까지 확대 표시합니다.
- 일부 좌우 이미지는 크롭될 수 있지만, 작은 글씨 가독성을 우선했습니다.
- 본문 CTA는 상·중·하 3개 유지합니다.

## 원본보다 크게 보이도록 극단 확대
- 이미지 안 작은 글씨는 이미지 자체 확대가 필요하므로 PC 155~170vw로 확대했습니다.
- 모바일은 280~320vw까지 확대했습니다.
- 일부 좌우 내용이 잘릴 수 있지만 작은 글씨 가독성을 우선합니다.
- 본문 CTA는 상·중·하 3개만 유지합니다.
- 이미지 생성은 하지 않았고 HTML/CSS 표시 방식만 수정했습니다.

## 좌우 잘림 완화 균형 확대
- 이전 280~320vw 극단 확대를 줄였습니다.
- PC는 116~126vw 중심으로 조정했습니다.
- 모바일은 155~182vw 중심으로 조정했습니다.
- 글씨는 조금 작아지지만 좌우 잘림을 줄이는 균형형입니다.
